# Modern Animated Auth UI

A modern and responsive authentication interface built with **HTML, CSS, and JavaScript**, featuring smooth animated transitions between Sign In and Sign Up forms.

---

## ✨ Features

- Smooth sliding animation between login and signup
- Responsive design for all screen sizes
- Social login UI buttons (Google, Facebook, GitHub, LinkedIn)
- Clean and modern UI design
- Easy to customize

---

## 🚀 Technologies Used

- HTML5
- CSS3 (Flexbox, Animations, Transitions)
- JavaScript (DOM Manipulation)
- Font Awesome Icons

---

## ⚙️ How to Run the Project

### 1. Clone the repository
```bash
git clone https://github.com/your-username/modern-animated-auth-ui.git
```
### 2. Go to the project folder
```bash
cd modern-animated-auth-ui
```
### 3.Open **index.html**  in your browser
